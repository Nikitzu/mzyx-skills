description: Turn a rough idea into an approved, decision-complete plan before writing any code.
---

Invoke the `mzyx-skills:think` skill.

Use this when the problem is fuzzy or the design is not yet locked. The skill challenges the problem, pressure-tests the design, and produces a plan another agent (or `/build`) can execute.

Modes:
- **Lightweight** — single-fix problems, returns a 2–3 sentence recommendation
- **Evaluation** — value/keep/remove judgments
- **Full** — multi-option, decision-complete plan

Output is a plan only. No code, no scaffolding. Pairs with `/spec` for structured specs and `/build` for execution.
