---
description: Produce distinctive, production-grade UI with a committed visual direction. Handles screenshot-driven iteration.
---

Invoke the `mzyx-skills:design` skill.

Use when building a frontend interface, component, or page — or when iterating on a UI from a screenshot complaint ("this looks wrong", "fix this"). The skill resists generic defaults, locks a visual direction, and produces production-ready output.

Pairs with `mzyx-skills:frontend-ui-engineering` (engineering rigor) — this command owns the visual direction; the other owns the engineering execution.
